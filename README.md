## Installation

1. Clone the repository:

```
git clone [https://github.com/hamdyhaggag/Job-Search-UI.git])
```

2. Navigate to the project directory:

```
cd Job-Search-UI
```

3. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```
